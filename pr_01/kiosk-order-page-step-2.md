# 키오스크 주문 웹페이지 2단계

## 1. 단계명

- 2단계: 공통 UI 스타일 작업

---

## 2. 단계 목표

- 화이트 베이스의 공통 디자인 방향을 실제 화면에 반영한다.
- 버튼, 카드, 레이아웃, 타이포그래피의 기준을 통일한다.
- 이후 기능 구현 단계에서 재사용 가능한 UI 스타일 구조를 마련한다.

---

## 3. 진행 상태

- 상태: `[x] 진행완료`

---

## 4. 핵심 작업 항목

### 4-1. 화이트 베이스 디자인 시스템 정의
- 메인 배경, 서브 배경, 라인 컬러, 포인트 컬러를 CSS 변수로 정의한다.
- 밝고 정돈된 인상을 유지하면서 대비가 충분한 텍스트 체계를 구성한다.

### 4-2. 공통 타이포그래피 규칙 정리
- 제목, 본문, 보조 텍스트의 크기와 굵기를 구분한다.
- 키오스크 화면 특성에 맞게 가독성이 높은 큰 타이포를 사용한다.

### 4-3. 공통 버튼 스타일 작성
- 주요 버튼과 보조 버튼 스타일을 분리한다.
- 터치에 적합한 여백과 둥근 형태를 적용한다.
- hover, active, focus 상태를 통일한다.

### 4-4. 공통 카드 및 레이아웃 스타일 작성
- 메뉴 카드, 패널 카드, 사이드 영역의 시각적 규칙을 통일한다.
- 섹션 간 위계가 보이도록 여백과 반경, 그림자 강도를 정리한다.

### 4-5. 카테고리 영역 및 메뉴 카드 스타일 구성
- 선택된 카테고리 상태가 명확히 보이도록 스타일링한다.
- 메뉴 카드의 가격, 설명, 강조 뱃지가 한눈에 읽히도록 정리한다.

- 공통 레이아웃 스타일 정의

- 메뉴 카드와 카테고리 버튼의 활성 상태 스타일 반영 완료
