# 키오스크 주문 웹페이지 개발 스케줄

## 1. 일정 개요

- 프로젝트명: 키오스크 주문 웹페이지
- 개발 언어: JavaScript
- 디자인 방향: 화이트 베이스의 깔끔한 UI
- 진행 방식: 핵심 기능 중심의 단기 일정
- 목표: 기획서 기준 필수 화면과 주문 흐름을 우선 완성
- 기준일: 2026-04-14
