# 키오스크 주문 웹페이지 3단계

## 1. 단계명

- 3단계: 메뉴 목록 기능 구현

---

## 2. 단계 목표

- JavaScript 기반 메뉴 데이터 구조를 정리한다.
- 카테고리별 메뉴 목록이 동적으로 렌더링되도록 구현한다.
- 메뉴 카드 클릭 시 다음 단계인 옵션 선택으로 자연스럽게 이어지도록 연결한다.

---

## 3. 진행 상태

- 상태: `[x] 진행완료`

---

## 4. 핵심 작업 항목

### 4-1. 메뉴 데이터 구조 작성
- 카테고리, 메뉴명, 가격, 설명, 인기 여부, 옵션 정보를 포함한 객체 구조를 정의한다.

### 4-2. 카테고리 렌더링 구현
- 카테고리 목록을 JavaScript로 생성한다.
- 선택된 카테고리 상태가 UI에 반영되도록 처리한다.

### 4-3. 메뉴 목록 렌더링 구현
- 선택된 카테고리에 따라 메뉴 카드가 필터링되어 표시되도록 한다.
- 메뉴명, 가격, 설명, 뱃지를 카드에 출력한다.

### 4-4. 메뉴 카드 클릭 이벤트 연결
- 메뉴 카드 또는 버튼 클릭 시 옵션 선택 모달이 열리도록 연결한다.

### 4-5. 목록 상태 보완
- 현재 카테고리의 메뉴 수를 안내한다.
- 필터 결과가 없을 때 빈 상태 메시지를 제공한다.

---

## 5. 3단계 산출물

- JavaScript 메뉴 데이터 정의
- 카테고리 목록 렌더링 기능
- 메뉴 카드 렌더링 기능
- 메뉴 카드 클릭 이벤트 연결
- 메뉴 목록 상태 안내 UI

---

## 6. 구현 결과

- `script.js`에 메뉴/카테고리 데이터 구조 반영 완료
- 카테고리 선택 시 메뉴 목록 필터링 구현 완료
- 메뉴 카드에 가격, 설명, 인기/신규 뱃지 출력 완료
- 메뉴 카드와 버튼 모두 옵션 선택 모달과 연결 완료
- 메뉴 개수 표시 및 빈 상태 메시지 보완 완료

---

## 7. 완료 기준

- 메뉴 데이터가 구조화되어 있다.
- 카테고리별 목록 렌더링이 동작한다.
- 메뉴 카드 클릭 이벤트가 정상 동작한다.
- 다음 단계에서 옵션 선택 기능과 자연스럽게 연결 가능하다.

## 8. 다음 단계

- 옵션 그룹 선택 로직 정리

## 9. 한 줄 요약
